# 合并对话与菜谱推荐 — 进度预览

> 创建时间: 2026-06-12 | 状态: ✅ 已完成

## 1. 任务概述

将前端「对话」页（`ConversationPage`）与「菜谱推荐」页（`RecipeRecommender`）合并为单一页面：
- **上方**：菜谱推荐表单（用餐类型、热量、厨具、偏好、获取推荐）
- **下方**：对话交互区（消息列表 + 输入框 + 发送）
- **导航**：移除独立的「菜谱推荐」标签
