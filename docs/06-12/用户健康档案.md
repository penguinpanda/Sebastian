# 用户健康档案

> 更新时间: 2026-06-12

## 1. 设计动机

此前 HealthAgent 每次分析都需要前端传身高、体重等参数。建立持久化的用户健康档案后：

- 用户只需填写一次，后续健康分析自动读取
- HealthAgent 可根据 `health_goal`（减重/增肌/维持）调整建议
- 与 `MealHistory` 饮食历史协同，提供 7 日饮食趋势洞察

## 2. 数据模型

### `UserProfile` — `app/models/user_profile.py`

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | UUID | 主键 |
| `user_id` | String(64) | 用户 ID，唯一约束 + 索引 |
| `age` | Integer | 年龄 (1-120) |
| `gender` | String(10) | male / female / other |
| `height_cm` | Float | 身高 cm (50-250) |
| `weight_kg` | Float | 体重 kg (20-400) |
| `activity_level` | String(20) | low / medium / high |
| `health_goal` | String(40) | lose_weight / maintain / gain_muscle |

## 3. API 端点

### 3.1 创建/更新 `POST /api/profile`

**请求 (upsert):**
```json
{
  "user_id": "u1",
  "age": 25,
  "gender": "male",
  "height_cm": 175,
  "weight_kg": 70,
  "activity_level": "medium",
  "health_goal": "maintain"
}
```

**处理逻辑:**
1. 按 `user_id` 查询已有档案
2. **存在:** 更新所有字段
3. **不存在:** 创建新记录

**响应 (200):**
```json
{
  "user_id": "u1",
  "age": 25,
  "gender": "male",
  "height_cm": 175,
  "weight_kg": 70,
  "activity_level": "medium",
  "health_goal": "maintain",
  "updated_at": "2026-06-12T10:00:00+00:00"
}
```

### 3.2 查询 `GET /api/profile`

```
GET /api/profile?user_id=u1
```

**无记录时:**
```json
{"user_id": "u1", "detail": "not_found"}
```

## 4. 与 HealthAgent 的集成

`POST /api/agents/health/analyze` 在处理请求时自动读取 `UserProfile`：

```python
# app/api/routes/agent_tools.py → health_analyze()

profile = db.execute(
    select(UserProfile).where(UserProfile.user_id == payload.user_id)
).scalars().first()

if profile:
    # 如果请求未提供身高体重，自动使用档案中的值
    if profile.height_cm and payload.height_cm == 0:
        payload.height_cm = profile.height_cm
    if profile.weight_kg and payload.weight_kg == 0:
        payload.weight_kg = profile.weight_kg

    user_profile = {
        "age": profile.age,
        "gender": profile.gender,
        "height_cm": profile.height_cm,
        "weight_kg": profile.weight_kg,
        "activity_level": profile.activity_level,
        "health_goal": profile.health_goal,
    }
```
