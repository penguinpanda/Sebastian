# Sebastian 使用指南

更新时间: 2026-06-11

## 1. 快速启动

### 1.1 前置条件

- Python 3.12+
- Node.js 18+
- Docker Desktop
- conda（推荐）

### 1.2 后端启动

```powershell
conda create -p ./.conda python=3.12 -y
conda activate ./.conda
python -m pip install -e .[dev]
Copy-Item .env.example .env
docker compose up -d postgres redis
alembic upgrade head
.\.conda\python.exe -m uvicorn app.main:app --reload
```

默认地址: `http://127.0.0.1:8000`

### 1.3 异步任务（可选）

```powershell
.\.conda\python.exe -m celery -A app.workers.celery_app:celery_app worker --loglevel=INFO
.\.conda\python.exe -m celery -A app.workers.celery_app:celery_app beat --loglevel=INFO
```

### 1.4 前端启动

```powershell
cd frontend
npm install
npm run dev
```

默认地址: `http://127.0.0.1:5173`

前端后端地址通过环境变量配置：

```bash
# frontend/.env.development
VITE_API_BASE_URL=http://127.0.0.1:8000/api
```

## 5. 常见问题

1. readiness 返回 503
   - 检查 postgres/redis/elasticsearch 是否可连接。
2. agent chat 超时
   - 检查 `DEEPSEEK_API_KEY` 与网络。
   - 适当调高 `LLM_TIMEOUT_MS`。
3. 前端无法联通后端
   - 检查 `frontend/.env.development` 的 `VITE_API_BASE_URL` 与 CORS。
4. 集成测试失败
   - 确认 Docker 正常，先执行 `docker compose ps`。
