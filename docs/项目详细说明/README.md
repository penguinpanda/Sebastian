## 核心认知

新人需要建立的三个认知：

1. **这是 AI Agent 后端，不是 CRUD 项目** — 请求经过 LangGraph 编排
2. **Agent 是薄门面，Service 是大脑** — 改功能先找 Service
3. **有两套编排体系** — Inventory Chat（graph.py）和 Agent Tools（agent_graphs.py）
