## 各层职责

| 层 | 目录 | 职责 | 典型文件 |
|----|------|------|----------|
| API | `app/api/` | 接收 HTTP 请求、参数校验、HTTP 状态码转换 | `routes/agent_tools.py` |
| Agent | `app/agents/` | 业务能力入口，薄封装，委托给编排层 | `recipe_agent.py` |
| Orchestration | `app/orchestration/` | LangGraph 流程控制、节点串联 | `agent_graphs.py`, `graph.py` |
| Service | `app/services/` | 核心业务逻辑、LLM 调用、领域规则 | `recipe_agent_service.py` |
| Repository | `app/repositories/` | 数据访问抽象，隔离 SQL | `inventory.py` |
| Model | `app/models/` | SQLAlchemy ORM 表定义 | `inventory.py`, `agent.py` |
| Schema | `app/schemas/` | Pydantic 请求/响应模型 | `agent_tools.py` |
| LLM | `app/llm/` | 模型客户端、Prompt、输出解析 | `client.py`, `prompts.py` |

## 两条主链路

Sebastian 有两套编排体系，新人容易混淆：

### 链路 A：专用 Agent Tools（结构化 API）

```
POST /api/agents/recipe/recommend
  → RecipeAgent.recommend()
  → run_recipe_agent()          # agent_graphs.py
  → RecipeAgentService          # 业务 + LLM
  → (子图) SearchAgentService   # 记忆检索
  → (子图) EquipmentAgentService # 厨具校验
```

适用于：前端或 MCP 直接调用某个 Agent 能力。

### 链路 B：Inventory Chat（自然语言）

```
POST /api/agent/chat
  → run_inventory_agent()       # graph.py
  → normalize → classify_intent → compose/fallback
  → LLM (prompts.py)
```

适用于：用户自然语言对话，当前聚焦库存意图。

### 链路 C：Inventory CRUD（纯 Service）

```
POST /api/inventory
  → InventoryService
  → PostgresInventoryRepository
  → PostgreSQL
```

适用于：结构化库存管理，不经过 Agent。

## 数据边界

| 存储 | 用途 | 原则 |
|------|------|------|
| PostgreSQL | 库存、Agent 任务、工具调用日志、Celery 执行日志 | **事务真相源** |
| Elasticsearch | 记忆索引 `memory_index` | 检索专用，不承担事务一致性 |
| Redis | 限流窗口、任务状态缓存、Celery broker | 短期状态，非长期审计 |

## Agent 协作示例（Recipe 子图）

Recipe 是当前唯一的多节点 Agent 图：

```
START
  → collect_context    # 调用 Search 子图，检索记忆
  → compose            # RecipeAgentService 生成菜谱
  → check_equipment    # EquipmentAgentService 校验厨具
  → finalize           # 合并厨具校验结果到响应
  → END
```

源码：`app/orchestration/agent_graphs.py` 中 `_build_recipe_graph()`。

## 依赖注入模式

API 层通过 FastAPI `Depends` 组装依赖链：

```python
# agent_tools.py
def get_recipe_agent(db: Session = Depends(get_db_session)) -> RecipeAgent:
    inventory_service = InventoryService(repository=PostgresInventoryRepository(db))
    return RecipeAgent(service=RecipeAgentService(inventory_service=inventory_service))
```

Agent 本身不持有 DB Session；Repository 在 API 依赖注入时创建。
